OCI runtime-spec v. 1.0.2 release notice

Date: 3/27/2020

We are excited to inform you that OCI runtime-spec has released version 1.0.2!

The OCI Charter’s IP Policy, in Section 8.d., provides for notice of the release of a new version of the OCI specification to be provided to all Members. This notice is a reminder that each release triggers the obligations set forth in the Open Web Foundation Final Specification Agreement (OWFa 1.0) (Patent Only) for all Members on the date 30 days after the date of this notice. Links for your convenience:

  • GitHub links for release version 1.0.2: https://github.com/opencontainers/runtime-spec/releases/tag/v1.0.2
  • The OCI Charter and IP Policy: https://www.opencontainers.org/about/governance.
  • The OWFa 1.0 (Patent Only) license: http://www.openwebfoundation.org/legal/the-owf-1-0-agreements/owfa-1-0—patent-only
